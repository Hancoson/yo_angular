# kwoh5

## 目录结构

```
├── README.md               
├── bower.json
├── bower_components	
└── src                     //源码
    ├── scripts             //脚本
    │   ├── filters         //过滤器
    │   ├── layout          //页面结构
    │   ├── pages           //业务代码
    │   ├── ui              //ui组件
    │   ├── util            //工具代码
    │   └── vendor          //第三方（建议通过bower获取）
    └── styles              //样式
        ├── components      //组件样式
        ├── layout          //框架样式
        ├── pages           //业务样式
        └── vendor          //第三方（建议通过bower获取）
```
